# Media Processor
Media Processor 是一個極簡主義風格的一個圖像媒體編輯器 。

🛠 任何編輯都基於 “圖層”，在你更動相片之前，原始檔案不會被破壞。可以同時在現有的編輯下輕鬆的嘗試不同的結果，對於圖片的編輯 <ins>大大降低錯誤翻車的機率</ins>。

🧰 除了基本的圖像編輯處理，例如裁剪圖片、濾鏡，<ins>你還可以加入 “擴展“ 來增加功能！</ins>
